## 🚀 使用指南

### 快速开始（3 分钟）
```
1. 打开 editor.html
2. 在 "Tally Form Embed" 标签输入 Tally 链接
3. 点击 "Add Tally Section"
4. 看到预览中出现绿色虚线框 ✓
```

### 修改配置
```
1. 在 Sections 列表选中一个 Tally
2. 修改宽度和内边距
3. 点击 "Apply Settings"
4. 预览立即更新 ✓
```

### 导出 HTML
```
1. 所有配置完成后
2. 点击 "Export HTML" 按钮
3. 下载包含 Tally 的独立 HTML 文件 ✓
```
